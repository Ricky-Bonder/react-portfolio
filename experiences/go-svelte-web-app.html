Modern Full-Stack Platform

Architecting the migration from legacy monoliths to Go & SvelteKit.

The Overview

A strategic modernization initiative to replace legacy systems with a high-performance, cross-platform architecture. The solution runs natively on resource-constrained Embedded Linux devices while simultaneously serving a responsive web interface for remote management. Operating with total autonomy and reporting directly to management, I drove the full lifecycle—bridging the gap between Product (PO) requirements and User Experience (UX) designs to deliver a robust, scalable product.

Engineering Challenges

Legacy Migration & Performance

Replacing a heavy Java monolith with Go allowed us to reduce the memory footprint by 60% on limited hardware, while maintaining feature parity through a Domain-Driven Design (DDD) approach.

Unified UX Architecture

The UI required a 'native' feel on touch-screen embedded devices and responsiveness on desktop. SvelteKit + Tailwind allowed us to build a shared component system (documented in Storybook) for consistent UX across targets.
